# تحديث: نظام رواتب بالنسبة + قائمة رواتب منفصلة

## 1. فورم "إضافة معلمة" بقى مبسط
شلت منه: الراتب الأساسي، تاريخ القبض، المكافآت، الخصومات، الملاحظات.
باقي بس: الاسم، السن، المحافظة، رقم الهاتف، واتساب.

## 2. صفحة جديدة في القائمة الجانبية: "💰 قائمة الرواتب" (`/salaries/`)
فيها قسمين:

**أ) نسبة كل معلمة (قابلة للتعديل مباشرة من الجدول):**
- كل معلمة جديدة بتاخد افتراضيًا **60% ليها / 40% للمنصة** من إجمالي قيمة اشتراكات طلابها الحاليين.
- تقدري تغيري النسبة لأي معلمة في أي وقت من نفس الصف (رقم + زرار حفظ).
- أو تفعّلي "راتب مثبت" بدل النسبة تمامًا (لو معلمة شغالة بمرتب ثابت مش نسبة) - بمجرد ما تحطي علامة صح وتكتبي الرقم، الراتب بتاعها هيبقى ثابت لحد ما تشيلي العلامة تاني.

**ب) سجل عمليات الصرف الفعلية:**
- زرار "تسجيل صرف راتب" فوق الصفحة (وبرضه من صفحة أي معلمة نفسها) بيفتحلك فورم: تختاري المعلمة، تاريخ الصرف، مكافآت، خصومات، عدد أيام الإجازة، وملاحظات.
- **الراتب الأساسي بيتحسب تلقائيًا** وقت الحفظ من نسبتها (أو راتبها المثبت) في نفس اللحظة، والصافي النهائي = الأساسي + مكافآت - خصومات.
- كل عملية صرف بتفضل محفوظة كسجل تاريخي، تقدري تفلتري بالسنة/الشهر/اسم المعلمة، وتشوفي الإجمالي.

⚠️ **ملحوظة عن أيام الإجازة:** دلوقتي بتتسجل بس كمعلومة (للتوثيق)، مبتخصمش تلقائيًا من الراتب. لو عايزة كل يوم إجازة يخصم مبلغ معين تلقائيًا، قوليلي وأضيفها بسهولة.

## 3. صفحة "النسب" (`/statistics/`) اتظبطت
بقت بتعرض نسبة كل معلمة الفعلية (مش رقم ثابت للكل)، ولينك يوديكي لصفحة "قائمة الرواتب" لو عايزة تغيّري.

## 4. صفحة أي معلمة (`/teacher/<id>/`)
بتعرض دلوقتي: إجمالي اشتراكات طلابها، نظام حسابها (نسبة ولا مثبت)، الراتب المستحق حاليًا، وآخر عمليات صرف الراتب ليها.

## ⚠️⚠️ تحذير مهم جدًا قبل ما تعملي migrate

التحديث ده **بيشيل نهائيًا** الحقول القديمة دي من جدول المعلمات في قاعدة البيانات:
`salary` (الراتب الأساسي)، `bonus` (المكافآت)، `deduction` (الخصومات)، `hire_date` (تاريخ القبض)، `notes` (الملاحظات).

**لو عندك بيانات حقيقية مسجلة في الحقول دي دلوقتي:**
1. ادخلي على `/admin/` أو أي صفحة فيها المعلمات وصوّري/انسخي القيم دي قبل أي حاجة (خصوصًا لو فيه معلومات مهمة في "ملاحظات").
2. بعد كده اعملي:
```bash
python manage.py makemigrations
```
Django هيسألك تأكيد على حذف كل حقل من الحقول دي - اختاري `y` (نعم، احذف الحقل) لكل واحد فيهم، لأننا فعلاً مش محتاجينها بعد كده.

3. بعدين:
```bash
python manage.py migrate
```

4. وبعد الـ migrate، افتحي صفحة "قائمة الرواتب" وابدئي تظبطي نسبة كل معلمة (أو راتبها المثبت لو حد شغالة بدون نسبة).
